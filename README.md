# SmartCart
=================
A simple, lightweight transportation plugin for MineCraft

## Features
### Control Blocks
- Black Wool (spawn)
- Yellow Wool (destroy)
- Orange Wool (slow)
- Red Wool (elevator)

# Requirements
This plugin is built against the [Spigot](http://www.spigotmc.org) MineCraft server.  Your mileage may vary with other servers.

# Roadmap
